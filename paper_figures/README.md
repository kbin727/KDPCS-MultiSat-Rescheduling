# Paper Figures

This directory contains figure assets generated from the released experiment
records. They are provided for inspection and reproducibility of the reported
plots.

The Pareto-front subfigures are stored in
`pareto_final_all18_full_20260705/`.

The numerical sources for the experimental figures are stored under
`experimental_records/`.  The scripts used to regenerate the main experimental
figures are archived in `tools/figure_generation/`.
